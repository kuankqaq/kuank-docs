# 隐私政策

## SlimeVR FBT Server for Android 隐私政策

本应用不会收集或保留您的任何个人信息。但是，为了实现基本功能，它需要向其他系统发送一些信息。未经您的明确指令，数据不会发送到本地网络之外。

建立连接时，以下数据可能会发送到您选择的服务器：
* 来自连接到应用的传感器（如 SlimeVR 追踪器）接收到的信息；
* 您手机的加速度和旋转数据；
* 您手机的本地 IP 地址以及发送和接收网络数据包所需的其他数据；
* 您在应用设置中输入的任何其他数据。

此外，本应用包含自动发现功能，该功能会尝试在您的本地网络上自动寻找兼容的服务器和设备。当应用打开时，它会定期向整个本地网络广播 UDP 数据包。这些数据包包含您手机的本地 IP 地址以及发送和接收 UDP 数据包所需的其他数据。

本隐私政策不涵盖您所连接的服务器对您数据的使用。前述数据因此受您本地网络上兼容服务器（或您选择输入的其他网络 IP 地址所在网络）的隐私政策约束。
