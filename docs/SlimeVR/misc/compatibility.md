# 兼容的软件和游戏
SlimeVR 支持任何支持 SteamVR 全身追踪的 SteamVR 游戏。
一些热门选择包括：
- VRChat
- Chillout VR
- Resonite
- Blade & Sorcery
- Dance Dash
- Beat Saber（使用模组）
- Zenith VR
- 以及更多！

SlimeVR 也可以在独立头显上用于 VRChat，如上所列！



### VR 工具和估算

#### Virtual Desktop 估算
SlimeVR 兼容并经常与 Virtual Desktop 估算一起使用，特别是用于手臂！

#### Quest Inside-Out Body Tracking
Quest IOBT 通常与 Virtual Desktop 手臂估算结合使用！

#### OVR Advanced Settings
SlimeVR 与 OVR Advanced Settings 完全兼容，不会产生冲突。
