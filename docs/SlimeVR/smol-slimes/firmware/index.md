# Smol 固件

本部分概述了 Smol Slimes 追踪器和接收器所使用的固件，并提供了详细指南的链接。

## 本部分内容

- **[预编译固件](smol-pre-compiled-firmware.md)**
  下载链接和使用预构建固件镜像（适用于常见硬件和引脚配置）的说明。本部分还包括专门为 SlimeVR 追踪器和接收器编译的自定义引导加载程序（UF2 和 HEX）。

- **[刷写固件](smol-flashing-firmware.md)**
  在支持的板卡和接收器上刷写固件的说明，包括 UF2 和基于 Nordic 的引导加载程序方法。

- **[SmolSlimeConfigurator](SmolSlimeConfigurator.md)**
  在 Linux 和 Windows 上通过简单 UI 在一个应用中配对和校准您的 Smol Slimes。
