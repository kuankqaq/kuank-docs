# Smol 社区 DIY

探索社区创建的构建、绑带和底座，帮助您组装或定制自己的 Smol 追踪器。

## 本部分内容

- **[社区构建](../hardware/smol-slimes-community-builds.md)**
  展示社区提交的 Smol 追踪器构建，包括照片和构建说明。
  仅包含细节足够以可复现的构建。

- **[社区绑带](../hardware/smol-slimes-community-straps.md)**
  社区来源的设计和指南，用于创建舒适且牢固的 Smol 追踪器绑带。

- **[社区底座](./smol-slimes-community-docks.md)**
  社区来源的与 Smol 追踪器兼容的底座设计和指南。
