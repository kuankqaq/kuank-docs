# IMU 校准

请确保在打开追踪器时，它放置在平坦的表面上。传感器需要在稳定环境中校准 20-30 秒。每次打开追踪器时都应执行此操作，否则会导致漂移率增加。

某些 IMU，如 BNO085 或 ICM20948，不需要任何特定的手动校准，在静置后即可直接使用。

其他所有 IMU 都需要校准才能正常工作。此校准只需执行一次。您一次只能校准一个 IMU，因此如果您有任何扩展模块，它们也需要进行校准。请注意，使用扩展模块时，校准数据保存在主追踪器上，因此在主追踪器之间交换扩展模块需要重新校准。

## BMI160（固件 v0.3.2 及以下）、MPU9250 或 MPU+QMC5883L

1. 插入您的微控制器（D1 Mini、NodeMCU 或其他）。
2. 打开 SlimeVR 服务器，点击**设置**，然后在**实用工具**下点击**串行控制台**。
3. 将您要校准的 IMU 倒置，按下微控制器上的重置按钮或 SlimeVR 服务器中的重启按钮。您应该会看到一条消息，提示您需要将 IMU 翻转正面朝上以开始校准。
4. 翻转 IMU 后，校准应开始。要成功校准 IMU，您需要轻轻地在所有 3 个轴上旋转 IMU。
5. 大约 60 秒后，追踪器应成功校准，并开始在 SlimeVR 服务器中显示旋转数据。

## BMI160（固件 v0.3.3 及以上）、BMI270、LSM6DSO 和 LSM6DSV 或其他 sfusion IMU

如果您有 BMI160 且固件版本为 v0.3.3 或更高，或者您使用 BMI270、LSM6DSO、LSM6DSV 或其他 sfusion IMU，则需要以不同的方式校准 IMU，除非固件中另有说明：
1. 为获得最佳校准效果，建议将追踪器加热至正常工作温度。为此，请在开始校准前佩戴追踪器至少 20 分钟。
2. 要获取校准过程的额外反馈，您可以连接到串行控制台。这完全是可选的。
   * 插入您的微控制器（D1 Mini、NodeMCU 或其他）。
   * 打开 SlimeVR 服务器，点击**设置**，然后在**实用工具**下点击**串行控制台**。
3. 将您要校准的 IMU 倒置，按下微控制器上的重置按钮或 SlimeVR 服务器中的重启按钮。您应该会看到一条消息，提示您需要将 IMU 翻转正面朝上以开始校准。
4. 翻转 IMU 后，校准应开始。将 IMU 静置至少 15 秒。
5. 现在，将 IMU 放置在平坦表面上，依次放置剩余的 5 个面，等待串行控制台或 LED 指示您可以为每个面更换位置。
6. IMU 现在应已完成校准，并开始在 SlimeVR 服务器中显示旋转数据。

### BMI160 温度校准 (tcal)

可选地，您可以对 BMI160 IMU 执行温度校准，这是一个更高级的过程，但能显著减少漂移：

#### 使用 3D 打印机

1. 将您的追踪器放入冰箱或冷冻室一段时间，使其冷却至 15°C 以下。如果不确定追踪器的温度，您可以在追踪器开启时在 SlimeVR Server 中查看。
2. 打开追踪器并逐渐加热，如果温度低于 15°C，校准将自动开始。
   - 您可以使用类似 3D 打印机热床或热风枪之类的工具，但注意不要过热电池或熔化外壳。
   - 切记不要着急。温度校准应至少持续 15 分钟，否则可能导致部分校准不完整，从而增加漂移。
3. 当 IMU 达到 45°C 时，校准将完成并自动保存。

#### 使用吹风机

1. 将您的追踪器放入冰箱或冷冻室短时间（约 10 分钟），使其冷却至 15°C 以下。
2. 取出追踪器并将其放在稳定的表面上，如桌子。
   - 打开 SlimeVR 应用程序 -> 打开开发者模式 -> 在主页右下角，切换"更多信息"以查看关于您的追踪器及其温度的详细信息。
   - 打开它，让它自然达到工作温度；暂时不要使用吹风机！将追踪器靠近电脑可以更快达到更高温度。
3. 等待追踪器温度稳定。使用吹风机将其加热到约 43°C，暂时不要达到 45°C。
4. 让其自然冷却回工作温度。
5. 大约 30°C 时，使用吹风机将其加热到约 46°C，以完成 TCAL 的最后几个测量点，然后让其冷却。
6. 使用 [USB WebUI 检查工具](https://ilyasnow.github.io/TCAL-Checker/) 检查效果。

在温度校准过程中，追踪器可以移动，但在运动过程中不会记录任何数据。

可能很难判断校准过程的进展。在固件的 `defines_bmi160.h` 文件中将 `#define BMI160_TEMPCAL_DEBUG` 设置为 true，可以暴露更多关于该过程的信息，将 SlimeVR Server 中的常规温度读数替换为温度校准调试信息。
格式为 AXXYY，其中 A 是校准状态（1 - 未在校准模式，2 - 校准进行中），XX 表示校准进度（0 到 60），YY 是温度。完全温度校准的追踪器将显示为 160YY。
