# 备选 Wi-Fi 选项

## Windows 移动热点

您可以通过 Windows 的移动热点功能设置一个 Wi-Fi 热点。

1. 在 Windows 设置中导航至 [设置 > 网络和互联网 > 移动热点](ms-settings:network-mobilehotspot)。
2. 在"属性"部分，点击编辑，并将名称和密码设置为您想要的任何值。
   - 确保"频带"（又称"网络频带"）设置为 2.4 GHz，以供 SlimeVR 追踪器使用。热点只能是 2.4 GHz 或 5 GHz，不能同时支持两者，因此如果设置为"任意"或 5 GHz，则不会支持 2.4 GHz。
3. 配置好网络后，您可以打开"移动热点"开关。
