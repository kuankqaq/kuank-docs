| 参数              | Wi-Fi 追踪器 | Smol 追踪器                        |
| ---------------------- | ---------- | --------------------------------- |
| 追踪器尺寸           | 较大     | 约 40x50mm（含绑带支架） |
| 常规绑带宽度    | 50mm       | 30mm                              |
| 扩展追踪器     | 是        | 否                                |
| 平均追踪器重量 | 50g        | ~10-15g                           |

### 测量绑带长度

##### 所需工具
- 软卷尺  

##### 1. 测量放置区域
将绑带**贴合但不紧绷**地缠绕在每个[追踪器放置位置](#tracker-placement)周围。保持绑带平整水平。

##### 2. 增加余量
在每个测量值基础上增加 **10 cm**。长了可以剪短。

部分增加的长度用于通过卡扣和滑扣固定绑带。

##### 3. 最终检查
绑带应：
- 在运动中保持牢固。
- 不限制血液循环。
- 有 **5 cm** 的调节余地。

**示例：**  
5 米弹性绑带通常足以为一个穿着欧码 XL 的普通男性制作 6 条绑带。

## 🔒 卡扣

### 卡扣等级排名

排名标准：
- 在肢体上盲操作开合便捷性——能否在不看的情况下轻松在肢体上闭合？
- 站立时的舒适度——是否硌人或感觉笨重？
- 躺卧时的舒适度——躺下时能否感觉到？
- 对所握持绑带部分的摩擦力——绑带是否会滑动？
- 触觉标记——能否通过触觉正确分辨上下方向，防止绑带扭转锁定？

## ↔️ 滑扣

### 三滑道滑扣，又称织带滑扣

可调节绑带的小型但必要的组件。这个简单的装置由一个带分隔杆的矩形组成，形成两个开口。绑带从一个开口穿入，从另一个穿回，无需缝纫即可调节绑带松紧。与主卡扣配合使用效果最佳，以实现最大安全性。

### 滑扣等级排名

排名标准：
- 站立时的舒适度——最小化笨重感和夹捏感。
- 躺卧时的舒适度——表面光滑，无尖锐边缘。
- 对所握持绑带的摩擦力——牢固抓握，不会滑动。

## 🖨️ 卡扣和滑扣的打印材料

### 🟢 最佳性能

#### 🟢 PETG
- **强度**：良好的抗冲击性，比 PLA 更不易碎
- **耐久性**：能够很好地承受反复使用和汗水接触；通常比 PLA 具有更好的层间附着力，从而提高卡扣和其他承重打印部件的耐用性。
- **打印设置**：稍高温度打印以获得最佳层间粘合。PETG 通常需要很少或不需要部件冷却（仅在桥接时使用冷却）；较低的打印速度和适当的温度可提高强度。
- **问题**：PETG 具有吸湿性——它会吸收空气中的水分。存放和处理要保持干燥（使用干燥箱或在打印前干燥耗材）。水分会导致拉丝增加和打印缺陷。
- **可获得性**：最广泛可用的 FDM 3D 打印材料之一

### 🔴 长期使用后会破裂

**不推荐：** 这些材料在没有其他选择时可以使用。它们在工作性能和持久性关键属性方面存在问题。

#### 🔴 PLA+/PLA Pro
- **强度**：优于标准 PLA
- **耐久性**：🔴 一般；在反复应力下可能开裂
- **摩擦力**：比 PETG 摩擦力更高，滑扣可能卡滞
- **问题**：脆性，热敏感性（炎热房间/阳光）

#### 🔴 标准 PLA
- **强度**：最弱选项，容易断裂
- **耐久性**：🔴 对于需要反复使用的功能性部件来说较差，最好不用
- **摩擦力**：可用于滑扣但磨损更快
- **问题**：脆性，尤其是薄截面。对于 smol 外壳非常差
- **可获得性**：最易获得的 FDM 3D 打印材料之一
- **最适合**：在更好的材料打印前测试适配

## 绑带滑动故障排除

绑带滑动是自制绑带时最常见的问题之一。这会导致追踪器移动和旋转，降低追踪精度。为了良好的追踪效果，绑带不滑动至关重要。

以下是一些可以尝试防止或减少此问题的方法。

检查事项：
- 材料过于柔韧——较硬且抓力更强的材料抗滑动能力更好。
- 绑带太窄/太小。更宽的绑带能更均匀地分布压力，滑动更少。

思路：
- 在绑带朝向皮肤的一面添加几行热熔胶以增加抓力。
- 使用辅助绑带固定追踪器（如 [Tom Yum 的设计](./community-diy-straps.md#tom-yums)）。
