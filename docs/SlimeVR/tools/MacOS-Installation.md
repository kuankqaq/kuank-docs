# MacOS 安装

## 1. 下载

下载适用于 Mac 的 [SlimeVR](https://github.com/SlimeVR/SlimeVR-Server/releases/latest/download/SlimeVR-mac.dmg) 安装文件（.dmg）。

## 2. 安装

打开 .dmg 并将 SlimeVR 应用拖入应用程序文件夹。

## 3. 打开

从应用程序文件夹中打开 SlimeVR。（您将看到如下所示的错误）

![mac1](../assets/img/mac%201.png)

## 4. 安全性

前往系统设置 -> 隐私与安全性 -> 您应该会看到 SlimeVR 已被阻止，如下所示。点击"仍要打开"。

![mac4](../assets/img/mac%204.png)

## 安装 Java

根据您的处理器下载并安装 Java。

[M1 或更新型号](https://adoptium.net/temurin/releases/?version=17&os=mac&arch=aarch64&package=jre)

[x64（Intel）](https://adoptium.net/temurin/releases/?version=17&os=mac&arch=x64&package=jre)

## 允许追踪器连接到您的 Mac

前往系统设置 -> 隐私与安全性 -> 滚动到底部 -> 安全性 -> 允许配件连接：将其更改为"自动（当解锁时）"。

![mac2](../assets/img/mac%202.png)

## 允许网络访问

打开 SlimeVR 并按照设置指南连接追踪器。您可能会看到以下消息。点击"允许"以连接追踪器。

![mac3](../assets/img/mac%203.png)
