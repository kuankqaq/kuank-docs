# 移动端设置

有几种选项可以在移动设备上原生使用 SlimeVR，而无需使用 PC。本指南将帮助您在移动设备或 Quest 头显上设置 SlimeVR 服务器。

## Android

### 要求

- Android 8.0+（Oreo）
- 端口 8080 空闲
- 应用设置为在后台运行（<https://dontkillmyapp.com/>）

### 安装

从 Play 商店下载 [SlimeVR 应用](https://play.google.com/store/apps/details?id=dev.slimevr.server.android)以安装在 Android 上

您也可以从最新的 SlimeVR 发布版本下载 [SlimeVR-android.apk](https://github.com/SlimeVR/SlimeVR-Server/releases/latest/download/SlimeVR-android.apk) 并安装，特别是在没有 Play 商店访问权限的设备上。您可能需要启用从未知来源安装，通常您的手机将提示您执行此操作。
