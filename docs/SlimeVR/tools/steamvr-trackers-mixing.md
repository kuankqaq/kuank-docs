# 将 SteamVR 追踪器（Vive/Tundra）与 SlimeVR 混合使用

## 技术细节

SteamVR 追踪器（如 Vive 和 Tundra）确实可以在一定程度上与 SlimeVR 协同工作。

它们在 SlimeVR 服务器内基本上被视为 Slime 追踪器，意味着它们的位置数据目前会被忽略。

计划进一步提供对 Vive 位置数据的完整支持，但暂无具体时间表。

## 在 SteamVR 追踪器上添加 SlimeVR 肘部追踪器

有两种方法可以实现：

1. 从头显追踪，需要 1 个**胸部**追踪器 + 2 个**上臂**追踪器。
2. 从控制器追踪，仅需要 2 个**前臂**追踪器。

在执行此操作之前，请确保在 VRChat 的追踪设置中禁用了肩部追踪！

#### 从头显追踪（最舒适，便于轻松的膝盖追踪）

1. 确保 SlimeVR 追踪设置中的 `Force arms from HMD` **已启用**。
2. 在上臂和胸部佩戴并分配追踪器。
3. 启用肘部的 SteamVR 追踪器。*仅在 VRChat 中使用 lock-hip 或 lock-head 时，才启用胸部的 SteamVR 追踪器。如果您使用 lock-all，您可能希望将单独的 Vive 保留在髋部，因为这将为您提供更好的脊柱追踪精度（因为 SlimeVR 的胸部追踪器不如 Vive 精确）。*
4. 执行完全重置和安装重置。

#### 从控制器追踪（占用更少的追踪器）

1. 确保 SlimeVR 追踪设置中的 `Force arms from HMD` **已禁用**。
2. 确保 Feeder App 正在运行（如果没有，运行安装程序并确保勾选它）。
3. 在前臂佩戴并分配追踪器。
4. 仅启用肘部的 SteamVR 追踪器。
5. 执行完全重置和安装重置。

在 v0.9.1 及之前版本中，重置姿势是 i-pose（手臂笔直向下贴于身体两侧），当前的安装校准姿势是上臂向后，前臂和手向前。v0.10.0+ 将有设置来更改这些。

## 其他组合

几乎任何组合都是可行的，并且只要**您没有跳过某个身体部位**，都能获得不错的结果。

例如，"SlimeVR 胸部 + Vive 髋部 + SlimeVR 大腿 + Vive 脚部"是可行的。

"Vive 髋部 + 脚部，以及 SlimeVR 肘部（前臂）或 SlimeVR 胸部 + 肘部（上臂）"也同样可行（参见[上文](steamvr-trackers-mixing.md#在-steamvr-追踪器上添加-slimevr-肘部追踪器)）。

1. 确保 Feeder App 正在运行（如果没有，运行安装程序并确保勾选它）。
2. 将您的 Vive 追踪器分配到相应的身体部位。
3. 在 SlimeVR 的设置中，仅启用您想要添加的 SteamVR 追踪器。
4. 执行完全重置和安装重置。
